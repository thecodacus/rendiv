# @rendiv/skills

Internal package — AI skill definitions for rendiv. Not published to npm.

Contains structured knowledge files that teach AI coding assistants how to use the rendiv framework effectively. Follows the [Agent Skills specification](https://agentskills.io/specification.md).

## Structure

- `rendiv-video/SKILL.md` — Main skill manifest
- `rendiv-video/rules/` — Detailed topic-specific guidance
- `rendiv-video/assets/` — Example compositions
- `src/` — Runnable entry point for previewing examples in Studio
